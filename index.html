ArcadeGame2000 - Instructions

Controls

  • Move left - left arrow
  • Move right - right arrow
  • Move up - up arrow
  • Move down - down arrow

How to win

Using the controls noted above, move your player across the stone pathways until you get to the river!

Rules

  • Players receive 3 lives at the start of the game
  • Each time a player is hit by a bug, he/she loses a life
  • If a player loses all 3 lifes the game is over
